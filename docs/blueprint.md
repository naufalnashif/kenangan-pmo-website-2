# **App Name**: Visitor Insights

## Core Features:

- Visitor Tracking: Tracks each visit to the website, storing relevant data such as timestamp, device type, and page visited.
- Data Synchronization: Sync the visit data across all devices and platforms in real-time.
- Analytics Dashboard: Presents a dashboard visualizing website traffic, unique visitor counts, and visit patterns.
- Local and VPS Compatibility: Ensures the analytics feature functions without errors, whether deployed locally or on a Virtual Private Server (VPS).

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A), complementing the existing blue accents while maintaining a professional look.
- Background color: Light Gray (#F9FAFB), offering a clean and modern contrast to the dark text and blue accents.
- Accent color: Teal (#2DD4BF), for interactive elements such as buttons and chart highlights.
- Body and headline font: 'PT Sans', sans-serif, for clear and accessible typography across all devices.
- Use simple, outlined icons from a set like Heroicons to maintain a consistent visual language.
- A clean and organized layout with clear hierarchy, making it easy to understand the analytics data.
- Subtle transitions and animations to provide feedback and improve user experience.
